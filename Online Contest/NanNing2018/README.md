最后甚至java比c++跑得还快。。

# Problem I

#### qhb hcx

题目描述：给出两个数字，将它们做差，得到一个数字，试求该数字是否可以被9整除后得到的除数是否由完全相同 的一个数字组成。(数字长度小于100)

cx通过观察9的倍数数字的规律发现，除了第一个数字和最后一个数字，中间的数字都是9，开头和结尾数字加和为9。

但是要注意以下几种情形

* 有前导0
* 最后一位为0
* 该数字为回文的

在这之后，只需要实现一个使用char完成的减法即可

决定将该减法补充到我们的模板里面
