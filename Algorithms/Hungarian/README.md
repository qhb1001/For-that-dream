# Problem A

题目描述：对每个结点染色，要求最终构成二分图。以染色的结点为黑色，未染色的结点为白色。试问至少染多少个黑色结点，可以使得图成为二分图。

由于给定一个图，将其染成二分图的方案是一定的，只是自己要选那种颜色来染是不确定的

于是维护一个联通分量中两种颜色的数量，取最小即可

注意判定图不可以成为二分图的情况 — 出现了染色过的结点并且出现了冲突

第二步是跑一趟dfs求出目前所有结点到达关键点的路径上的最大值

第三步是寻找最优方案
